# Website Kasir Sederhana

## Deskripsi
Aplikasi kasir berbasis web untuk mencatat penjualan dan stok barang.

## Fitur
- Login Admin
- Tambah, edit, hapus produk
- Transaksi penjualan
- Riwayat Transaksi
- Cetak File

## Cara Install
1. Download project
2. Import database ke phpMyAdmin
3. Jalankan di XAMPP

## Teknologi
- PHP
- MySQL
- Tailwind CSS
